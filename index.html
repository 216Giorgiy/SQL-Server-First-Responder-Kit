SQL Server First Responder Kit

sp_Blitz, sp_BlitzCache, sp_BlitzIndex, and other SQL Server scripts for health checks and performance tuning.

Download this project as a .zip file Download this project as a tar.gz file

What Are These Scripts?

  • sp_Blitz - prioritized SQL Server health check to make sure your databases are backed up, you don't have corruption, and there's no ugly surprises in your server. Documentation
  • sp_BlitzCache - lists the most resource-intensive queries in your plan cache. Documentation
  • sp_BlitzIndex - checks a database's indexes for duplicates, heaps, missing indexes, and more. Documentation
  • sp_AskBrent - performance health check of what's broken on the server right now. Documentation
